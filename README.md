# Telecom-x-2
Telecom X – Predicción de Cancelación de Clientes (Churn)
Propósito del análisis

El objetivo de este proyecto es desarrollar modelos de Machine Learning capaces de predecir qué clientes tienen mayor probabilidad de cancelar los servicios de Telecom X. Esto permite a la empresa anticiparse al problema del churn y diseñar estrategias de retención más efectivas.

Estructura del proyecto

El proyecto se organiza de la siguiente manera:

TelecomX_Parte2.ipynb → Notebook con todo el proceso de análisis y modelado.

datos_tratados.csv → Dataset limpio utilizado para entrenar los modelos.

README.md → Descripción general del proyecto.

Dentro del notebook se incluyen las siguientes etapas:

Carga y preparación de datos

Codificación de variables categóricas

Balanceo de clases

Análisis de correlación

Entrenamiento de modelos

Evaluación y conclusiones

Ejemplos de insights obtenidos

Durante el análisis se identificaron algunos factores relacionados con la cancelación:

Los clientes con contratos mes a mes presentan mayor probabilidad de cancelar.

Los clientes con mayor cargo mensual tienen más tendencia al churn.

Los clientes con menos tiempo en la empresa presentan mayor riesgo de cancelación.

Estos insights pueden ayudar a la empresa a diseñar estrategias de fidelización.
